# Priced-Landing-Page

Belajar slicing menggunakan react

https://codedesign.dev/challenge/priced

## Installation

Install Priced-Landing-Page with npm

```bash
  git clone https://github.com/alipnf/Priced-Landing-Page.git
  cd Priced-Landing-Page
```

```bash
  npm instal
```

```bash
  npm run dev
```
